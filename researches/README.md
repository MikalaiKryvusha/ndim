# `researches/` — база знаний по большим трудным вопросам

Заметки и выжимки по крупным, сложным вопросам — исследовательские отчёты, накопленное ноу-хау, важная
справочная информация, на сбор которой ушёл реальный труд и которая не должна испариться. Один
`NN_<тема>.md` на тему исследования.

**Для человека (владельца):** место, где можно найти глубокий фон за принятыми решениями — почему
выбран тот или иной подход, что выяснилось про трудную проблему. Можешь посеять тему, которую хочешь
исследовать.

**Для ИИ-агента:** когда вопрос достаточно велик, чтобы его выводы пережили текущую задачу, — пиши
сюда исследовательскую заметку (сырые источники → анализ → выводы и гипотезы). Ссылайся на неё из
бага, плана или идеи, которые её породили (DRY — не исследуй дважды). Исследовательская заметка —
**живой справочник**, а не закрываемая задача: не помечай её `DONE`, продолжай обновлять по мере
роста понимания.
