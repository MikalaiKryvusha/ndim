## 2. Что такое проект (изучено)

**NDim Space** — веб-приложение для поиска похожих людей: альтернатива индустрии знакомств,
построенная не на удержании и подписке, а на честном математическом поиске близких по духу.

### Главная математическая идея (ядро)

Каждый пользователь описывает себя набором **измерений** (`dims`) — произвольных осей, которые
предлагает сообщество, — и ставит себе по каждой оценку 0…10 («звёзды»). Пользователь = точка в
своём подпространстве измерений.

Связь двух пользователей (`legacy/storage/admin/adminFunctions.js` → `calculateRelation`):

```
общие оси      K = dims(owner) ∩ dims(guest)
расстояние     d = √( Σ_{k∈K} (owner[k] − guest[k])² )        # евклидово в общем пространстве
диаметр        D = √|K| · 10                                   # диагональ K-мерного куба со стороной 10
близость       proximity   = 1 − (d / D)^0.7                   # ^0.7 — повышенная чувствительность вблизи
общность       commonality = 2|K| / (|dims(owner)| + |dims(guest)|)   # коэффициент Дайса
ПОХОЖЕСТЬ      similarity  = proximity × commonality
```

Плюс ~20 производных метрик пространства (диаметры, доли пересечения), сохраняемых под
обфусцированными именами `a…t`.
