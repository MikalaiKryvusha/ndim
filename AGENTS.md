# Правила для агента — NDim Space

Этот проект обёрнут фреймворком **KAIF**. Рабочий язык — **русский**.

Перед каждой задачей прочитай `AGENT_GUIDE.md` (канон) и `STATUS.md` (текущее состояние); думай по
`PHILOSOPHY.md` (KISS + бритва Оккама); чини дефекты по `BUG_FIXING_FRAMEWORK.md`; вспоминай уроки
грепом по `EXPERIENCE.md`.

Ключевые инварианты:

- **Старого кода здесь нет.** NDim Space 1.x целиком живёт в приватном репозитории
  `MikalaiKryvusha/ndim-old`. Его знание выжато в `researches/02` (модель данных Firestore) и
  `researches/03` (математическое ядро и матрица соответствия `a…t`).
- **Цель проекта — добро, а не выручка** (`GOAL.md`). Механики удержания и подписки противоречат ей.
- **UI/UX и бренд агент не решает один:** 4 варианта макета → утверждение автором. Всё остальное —
  через `interviews/`.
- **Перед каждым push** — `npm run kaif:check` (проверка на секреты и ПДн в git).

Скиллы (ритуалы работы) лежат в `.claude/skills/<имя>/SKILL.md` — по одной `/команде` на скилл.
Для агентских систем, читающих `.roo/commands/` или иной формат, переведи их механически: содержание
важнее пути.
