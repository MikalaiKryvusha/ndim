# Bug 45 — Нет управления аккаунтом: удалить аккаунт и сменить почту невозможно

**Статус:** 🔴 OPEN (🔴 приоритетно: обещание «человек может удалить свои данные» сейчас
невыполнимо; смена пароля — вместе с bugs/32, пачка 4) ·
**Источник:** полный проход инвентаря паритета (researches/12, 2026-07-24)

## Симптом
В 1.x с «Дома» вела дверь «Управлять аккаунтом» (кадр app-14). В 2.0 у человека есть только
«Выйти» в меню. Ни удалить аккаунт, ни сменить почту нельзя — а канон текста продукта
запрещает слово «навсегда» ИМЕННО потому, что «человек может удалить свои данные».

## Канон 1.x (app-14 + edit_email/edit_password/delete_account)
Экран «Управлять аккаунтом»: карточка «Мой аккаунт» (**Email** · **Создан** дата+время)
и три двери:
- «Поменять пароль» — текущий + новый ×2, глаз 👁;
- «Поменять Email» — новый email + текущий пароль → письмо-подтверждение;
- «Удалить аккаунт» — «Ваша учётная запись и все её данные будут безвозвратно удалены.
  Удалить?» + текущий пароль + красная кнопка «Удалить» / «Отмена».

## Адаптация к 2.0 (вход без пароля)
Подтверждение личности — не паролем, а **свежей реавторизацией** (повторная ссылка на почту /
Google `reauthenticateWithPopup`); смена почты — `verifyBeforeUpdateEmail`. Удаление обязано
снести ВСЁ: Auth-пользователя, документы Firestore (профиль/бакеты/оценки/relations),
Storage-аватар — и не тронуть чужие топы (их вычистит суточный проход сервера синхронизации —
проверить, что чистит удалённых, а не только гостей).

## Развилка владельцу (не блокирует разведку и каркас)
Экран — самое очевидное по канону app-14 (карточка + строки-двери), но подтверждение
удаления в беспарольном мире — формулировку выбрать владельцу (двойной ввод слова «УДАЛИТЬ»?
только реавторизация?). Включить одним вопросом в интервью пачки 4.

## План проверки
Стенд: удаление → вход той же почтой создаёт ПУСТОЙ новый аккаунт; оценки/бакеты/аватар
удалены (тест); сервер синхронизации выкидывает удалённого из чужих топов; смена почты —
письмо приходит, старая почта перестаёт входить. Правила: никто не может удалить чужое.
