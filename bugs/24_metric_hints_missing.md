# Bug 24 — Пропали подсказки по метрикам Похожесть / Близость / Общность

**Статус:** 🔴 OPEN (пачка 3) · **Источник:** волна 09 (п. 6)

## Симптом (слово владельца)
«Пропали подсказки по метрикам Близость, Общность, Похожесть. В оригинальном NDim были и
вызывались тапом по метрике. Рядом с метрикой была иконка инфо, показывающая, что там есть
подсказка. Вернуть это в новом стиле».

## Что не так сейчас
На «Связях» тройка метрик — просто цифры; тап по метрике ничего не делает, иконки ⓘ нет.
Человек, не читавший руководство, не знает, чем Близость отличается от Общности.

## План фикса
1. Тексты подсказок НЕ выдумывать: определения всех трёх метрик сняты дословно в
   `researches/07_user_manual_1x.md` (§3, словарь) — взять оттуда, RU/EN.
2. UI: иконка ⓘ рядом с подписью метрики; тап по метрике/иконке раскрывает подсказку
   (поповер или разворот под тройкой — по месту, в стиле 2.0, обе темы).
3. Пройтись по остальным местам с этими метриками (раскрытая карточка связи, «Пространство»
   — средняя похожесть): подсказка должна быть везде, где метрика на виду.

## План проверки
Стенд, Playwright: тап по каждой из трёх метрик → подсказка видна, текст совпадает с
руководством; повторный тап/тап мимо — закрылась; консоль чиста. Скриншоты.
