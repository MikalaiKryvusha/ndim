# Домашка 04 — Прописать сайт в поисковиках (и включить www)

> Задача для человека: нужны твои аккаунты Google/Яндекс и твой DNS в Porkbun — агент туда
> не достаёт (и не должен). Всё техническое на стороне сайта уже готово: sitemap.xml, robots.txt,
> canonical, 404, Lighthouse 100/100.
> Заведена: 2026-07-12 · Основание: `researches/08_seo_indexing.md` §3, §9 (пп. 4, 9–11) ·
> `STATUS.md` → «Где продолжить», п. 1.

## ⏳ СТАТУС: ожидает владельца

Суммарно ~20 минут. Части независимы — можно делать по одной. Часть 0 — вообще одно слово в чат.

---

## Часть 0 — скажи агенту два слова (по минуте каждое)

1. **«Деплой лендинга»** — sitemap.xml, robots.txt и страница 404 собраны и проверены, но на
   живой домен их выкатывает команда деплоя, которую агент без твоего явного слова не запускает.
2. **«Делай www»** — агент через Hosting API заведёт `www.ndimspace.app` как редирект на
   `ndimspace.app` и сразу скажет, какие DNS-записи внести в Porkbun (обычно одна A-запись).
   Это тоже действие на живом домене — нужно твоё слово.

## Часть 1 — Google Search Console (~10 минут)

1. Открой **https://search.google.com/search-console** (аккаунт `kotkrinik@gmail.com`).
2. «Добавить ресурс» → тип **«Доменный ресурс»** (левая колонка) → введи `ndimspace.app`.
3. Google покажет TXT-запись вида `google-site-verification=...` — **скопируй её**.
4. В новой вкладке: **porkbun.com** → Domain Management → `ndimspace.app` → **DNS Records** →
   добавь запись: тип **TXT**, Host — *пусто* (корень домена), Answer — скопированная строка.
5. Вернись в Search Console → «Подтвердить». Если не подтверждается сразу — подожди 10–30 минут
   (DNS расходится не мгновенно) и нажми ещё раз.
6. После подтверждения: раздел **«Файлы Sitemap»** → добавь `https://ndimspace.app/sitemap.xml`.
7. Вставь в строку проверки URL вверху `https://ndimspace.app/` → «Запросить индексирование».

> TXT-запись Google просит хранить постоянно — не удаляй её потом.

## Часть 2 — Яндекс.Вебмастер (~5 минут)

1. Открой **https://webmaster.yandex.ru** (любой твой Яндекс-аккаунт, например от
   `kotkrinik@yandex.ru`).
2. «Добавить сайт» → `https://ndimspace.app`.
3. Способ подтверждения — **DNS-запись**: Яндекс покажет TXT вида `yandex-verification=...` —
   добавь её в Porkbun так же, как в части 1 (тип TXT, Host пусто).
4. После подтверждения: «Индексирование» → «Файлы Sitemap» → добавь
   `https://ndimspace.app/sitemap.xml`.

## Часть 3 — Bing (~3 минуты, после части 1)

1. Открой **https://www.bing.com/webmasters** → войди тем же Google-аккаунтом.
2. Выбери **«Импортировать из Google Search Console»** — верификация и sitemap подтянутся сами.

## Что будет дальше (работа агента, без тебя)

- После «деплой» — агент выкатит артефакт, проверит sitemap/robots/404 на живом домене.
- После «делай www» — агент создаст редирект и продиктует точные DNS-записи.
- Когда GSC подтверждён — агент сможет следить за индексацией по мере появления новых страниц.
