# `homeworks/` — задачи от агента человеку

Задачи, о выполнении которых агент просит **человека** — то, чего он не может сделать сам в силу своей
цифровой, бестелесной природы: протестировать на реальном железе, действовать в физическом мире,
воспользоваться аккаунтом или доступом, который есть только у человека, что-то купить, наблюдать
что-то офлайн. Каждый документ описывает задачу конкретными шагами для человека и собирает обратно его
наблюдения и результаты. По одному `NN_<имя>.md`.

**Для человека (владельца):** когда агент завёл домашку, ему нужна рука в физическом или офлайн-мире.
Пройди шаги и запиши в документ, что ты увидел, — агент прочитает твои заметки и продолжит.

**Для ИИ-агента:** когда ты заблокирован на чём-то, что может сделать только человек с телом, — не
стой: напиши сюда домашку с ясными, минимальными пронумерованными шагами и местом для результатов
человека, затем продолжай с другой работой. Когда человек отчитается, учти результаты и пометь файл
тегом `DONE` (`git mv NN_x.md NN_DONE_x.md`).
