Решение интервью №003 (В1): первый экран 2.0 — профиль с измерениями. Состав — из модели данных 2.0: личные свойства с аудиторией по кругам (умолчание — «Никому») и оценки по осям 0–10, которые не видит никто, кроме Вас.
