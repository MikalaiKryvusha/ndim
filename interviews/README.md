# `interviews/` — решения уровня владельца

Интервью, которые агент проводит с человеком, чтобы закрыть решения, которые он **не имеет права**
принимать один: UI/UX, серьёзные технические развилки, бренд, видение, приоритеты. Закрытые вопросы
A/B/C/D с рекомендацией первой, на которые человек отвечает **прямо в документе**. По одному файлу
`interview_NNN_<тема>.md`.

**Для человека (владельца):** когда агент завёл интервью — он ждёт **тебя**. Заполни поля
«**Ответ:**» прямо в документе (выбери A/B/C или напиши своё в D). Здесь фиксируются и сохраняются
твои судьбоносные решения.

**Для ИИ-агента:** заводи интервью только для настоящих развилок уровня владельца (скилл:
`/interview`). Варианты — **A/B/C/D**: **A** всегда тот, что выбран через призму `PHILOSOPHY.md`
(простейший и самый эффективный), и помечен **(рекомендуется)**; **D** всегда «свой ответ» для
владельца. Сначала сделай подготовительную работу, держись в пределах 1–5 вопросов, затем
приостановись и дай владельцу ответить. Всё, что дёшево откатить, — решай сам.
