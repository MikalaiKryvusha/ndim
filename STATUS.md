### Что сделано в этой сессии (2026-07-12, вечер) — экран «Пространство»

Реализован по утверждённому V1 «Приборная панель» (`design/space-mockups.html`).
Смотреть: `npm run stand` → http://localhost:5173/space.

- **Сервер синхронизации теперь производит статистику Пространства** (`calculator/index.mjs`):
  пишет `space/stats` (люди · измерения · диаметр · оценки · связи · средняя похожесть ·
  распределение · новые измерения за сутки), снимок дня `space/stats/daily/{дата}` и сердцебиение
  с версией — `space/server`. Считает всё **чистый модуль** `src/lib/model/stats.ts` — тот же,
  которым экран эти цифры читает.
- **Тренды растут из снимков** (ideas/06): «за неделю» = разность с ближайшим снимком не позже
  недели назад; виджет «Сегодня» = разность со вчерашним. **Истории нет → тренда нет** (нулевой
  рост не выдумываем). Спокойные сутки → честное «Сегодня в Пространстве ничего не изменилось».
- **Состояние сервера ВЫВОДИТСЯ из сердцебиения, а не рисуется**: молчит дольше трёх интервалов —
  «Не отвечает» (слово выбрал агент: макет этот случай не покрывал; «остановлен» было бы враньём —
  мы знаем лишь, что он не отвечает). Зелёная лампочка — рядом со **словом состояния**, как велел
  владелец.
- **Стенд стал живым**: `npm run stand` теперь гоняет сервер синхронизации **службой** (цикл 15 с,
  `tools/stand.mjs`), а не однократно. Иначе стенд врал бы о продукте («Не отвечает» через 3 минуты),
  а оценка из браузера не долетала бы до «Связей» и «Пространства» сама.
- **Версии на виду**: приложение — `__APP_VERSION__` из `package.json` (vite `define`, версия
  проекта поднята до **2.0.0**); сервер — из `space/server`.
- **Правила**: клиенту запись в `space/stats` / `space/server` / снимки **запрещена** — цифры
  витрины не должен уметь подделать никто. Читают все вошедшие, **включая гостя** (в агрегатах
  нет ничего персонального).
- **Гость не входит в счётчики Пространства** — он невидим другим (В3): иначе цифра обещала бы
  людей, которых не найти.
- Побочно: русская морфология вынесена в `src/lib/ui/format.ts` (одна на все экраны), в токены тем
  добавлены `--up` / `--down` (тренды).
