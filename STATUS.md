Версия 1.x (весь фронтенд в одном `index.html` на 3083 строки плюс `app.js` на 9127 строк, Firebase, и

(30 тестов). Оба набора проверены **мутациями**: на сломанном коде краснеют.

Зависимостей у кода нет: **Node 24 исполняет TypeScript нативно**, тесты — встроенный `node:test`.
`tsc` и `@types/node` нужны только для `npm run typecheck`.

Модель данных 2.0 спроектирована (`researches/04`), интервью №002 закрыто. Профиль разбит на
документы-бакеты по кругам владельца — это **вынужденное** следствие двух фактов: Firestore не умеет
скрывать поля при чтении, а бэкенд без входящих портов не может отфильтровать их на лету.

`npm run e2e` — Playwright, 10 проверок на продакшен-сборке, включая пререндер и персистентность
темы/языка. Стенд проверен мутацией: на сломанном дефолте темы честно краснеет.
